# 华理音乐分享网站

## 项目简介

这是一个仿QQ音乐风格的华理音乐分享网站，由华东理工大学计算机职业实践小组第二小组开发的大一课程项目。网站提供了音乐播放、歌手展示、歌单推荐等基础功能，是一个音乐分享平台的雏形。

## 项目特点

- 🎵 仿QQ音乐风格的界面设计
- 🎸 多页面导航：音乐馆、歌手、歌单、歌曲、登录
- 📱 响应式布局设计
- 🎨 现代化UI界面，包含轮播图、卡片展示等元素
- 🔍 搜索功能
- 🎧 音频播放功能

## 文件结构

```
项目根目录/
```

## 功能模块

### 1. 首页 (index.html)
- 轮播图展示
- 歌单推荐
- 新碟首发
- 精彩推荐

### 2. 歌手页面 (singer.html)
- 歌手分类筛选（地区、性别、风格）
- 歌手卡片展示
- 动态筛选功能

### 3. 歌单页面 (song-list.html)
- 按地区分类：内地、港台、欧美、日本、韩国
- 歌单卡片展示
- 交互式切换

### 4. 歌曲页面 (song.html)
- 排行榜展示
- 歌曲列表
- 音频播放器

### 5. 登录页面 (login.html)
- 用户登录/注册表单
- 基础表单验证

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **UI组件**: Swiper轮播图
- **交互**: jQuery (歌手页面)
- **后端**: Java Servlet (登录验证)


## 使用说明

1. 直接打开 `index.html` 即可访问网站首页
2. 通过顶部导航栏可在不同页面间切换
3. 歌手页面支持按地区、性别、风格筛选
4. 歌单页面支持按地区分类查看
5. 歌曲页面包含音频播放功能

## 注意事项

- 本项目为教学演示用途，部分功能为静态展示
- 登录功能包含后端验证代码示例
- 音频文件路径可能需要根据实际部署环境调整

## 开发环境

- 推荐使用现代浏览器访问（Chrome、Firefox、Edge等）
- 文本编辑器：VS Code、Sublime Text等
- 本地服务器环境（如需测试登录功能）
